# Weather App 🌤️

A Python application that fetches real-time weather data for any city in the world.

## Features
- Get current weather for any city
- Check multiple cities without restarting
- Simple and easy to use

## How to Run

1. Install requests library:
pip install requests

2. Run the app:
python app.py

3. Enter any city name and get the weather!

## Technologies Used
- Python
- wttr.in API
